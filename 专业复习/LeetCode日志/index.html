每日温度

C++语法复习

  • vector< int > t
    • 属性: t.size()
    • 函数：vector< int > next_day(n); 创建一个大小为 n 的 vector< int > 数组

      解题思路

  • 单调栈
    • 栈中存储元素的下标，维持一个每日温度单调递减的栈
    • 遍历数组，当遇到比栈顶元素大的元素时，栈顶元素出栈，并计算出当前元素与栈顶元素之间的距离
    • 遍历完成后，栈中剩余的元素即为无法找到对应元素的下标，其对应的下一天位置为初始化的 0
  • 参考 栈进阶数据结构
